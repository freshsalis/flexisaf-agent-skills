# Final Delivery Summary

At the end of a completed task, provide a concise summary containing:

- Task completed.
- What was implemented.
- Files changed.
- Whether a database migration was added.
- Automated tests added or updated.
- Feature-specific test result.
- Full test suite result.
- Manual endpoint testing result.
- Database used for manual endpoint testing.
- PR review result.
- Suggestions, if any.
- Branch name.
- Commit message.
- Commit hash, if committed.
- Push status, if pushed.
- Any remaining risks or follow-up notes.
